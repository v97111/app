Tips

  • By default the dev server runs in LAN mode to avoid interactive tunnel prompts.
  • Set EXPO_LAUNCH_MODE=tunnel (and ensure @expo/ngrok is available) if you want Railway to create a tunnel automatically.
  • Otherwise run npx expo start --tunnel locally, copy the Expo Go link, and set it as EXPO_URL in Railway.
  • For published apps, use your project page like https://expo.dev/@ACCOUNT/SLUG?service=expo-go.
  • Make sure your phone has Expo Go installed.
